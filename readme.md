Vesikalık fotoğraf yerleştirme ve baskıya hazırlama programı. 

Kağıt boyutu seçimi, fotoğraf boyutu seçimi ve basılacak fotoğraf adet seçimi ile istediğiniz şekilde vesikalık fotoğraf baskısı yapabilirsiniz. Ayrıca kırpma özelliğini kullanarak fotoğrafınızın istediğiniz kısmını seçebilirsiniz.
